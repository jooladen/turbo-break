# 초딩용 ORB 가이드

> "ORB가 뭔데? 어떻게 쓰는 건데?" 여기서 다 해결합니다.

---

## 1. ORB가 뭔데?

**ORB = Opening Range Breakout = 오프닝 레인지 돌파**

어렵게 느껴지지만 사실 엄청 단순합니다.

### 핵심 아이디어 (딱 3줄)

```
1. 장 열리자마자 처음 30분 동안 주가가 오간 범위를 기록한다
2. 그 범위 위로 올라가면 → "오를 것 같다" (LONG 신호)
3. 그 범위 아래로 내려가면 → "떨어질 것 같다" (SHORT 신호)
```

### 비유로 이해하기

축구 경기 시작 후 처음 15분을 보면 오늘 경기 흐름이 보이죠?
ORB도 똑같아요. **장 시작 후 첫 30분을 보면 오늘 주가 방향이 보입니다.**

---

## 2. VWAP은 뭔데?

**VWAP = 거래량 가중 평균 가격**

그냥 "오늘 이 주식이 평균적으로 얼마에 거래됐나"예요.

- 현재가 > VWAP → 평균보다 비싸게 거래 중 → 강세
- 현재가 < VWAP → 평균보다 싸게 거래 중 → 약세

**프로 기관 투자자들이 VWAP 기준으로 매매합니다.** 이 라인 위에서 사는 게 훨씬 안전해요.

## 4. 신호 읽는 법

### 🟢 LONG 신호가 뜨면

| 항목 | 의미 |
|------|------|
| 진입 시간 | 이 시간에 ORB 위로 돌파함 |
| 진입 가격 | 돌파한 봉의 종가 |
| 거래량 X배 | 평소보다 X배 더 많이 거래됨 (클수록 강한 신호) |
| VWAP 위 ✅ | VWAP 위에서 돌파 = 더 믿을 수 있는 신호 |

### 🔴 SHORT 신호가 뜨면

반대예요. ORB 저가 아래로 주가가 깨진 것. 하락 가능성.

### 신호가 없으면

"ORB 돌파 시그널 없음 — 레인지 내 횡보 중"

→ 오늘은 방향이 안 잡힌 날. 그냥 기다리세요.

---

## 5. ORB 기간 어떻게 선택해?

| 기간 | 특징 | 추천 상황 |
|------|------|-----------|
| **5분** | 신호 빠름, 가짜 신호 많음 | 변동성 큰 날, 빠른 대응 |
| **15분** | 균형 잡힌 설정 | 일반적인 날 |
| **30분** | 신호 느림, 신뢰도 높음 | 안정적인 진입 원할 때 ✅ |

**모르겠으면 30분 쓰세요.** 가장 검증된 설정입니다.

---

## 6. 언제 사야 해?

ORB 신호는 **방향성만** 알려줍니다. 실제 매수는:

```
✅ 좋은 진입 조건
- 🟢 LONG 신호 + 거래량 2배 이상
- VWAP 위에서 돌파
- 일봉 스크리너에서도 걸린 종목

⚠️ 조심해야 할 상황
- 거래량 1.5배 미만 (약한 신호)
- VWAP 아래에서 돌파
- 장 막판 (15시 이후) 신호
```

## 7. 워크플로우 정리

```
1️⃣  /screener 접속 → 일봉 스크리너에서 관심 종목 발굴

2️⃣  해당 종목 행의 [📊 ORB] 버튼 클릭

3️⃣  분봉 차트에서 ORB 고가/저가와 VWAP 확인

4️⃣  시그널 카드 확인:
    - 🟢 LONG + 거래량 2배+ + VWAP 위 → 매수 검토
    - 신호 없음 → 오늘은 패스

5️⃣  실제 매수는 본인 판단으로! (이 앱은 참고용)
```
