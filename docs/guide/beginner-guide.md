# 초딩용 가이드: N일 고가 돌파 스크리너

> "이게 뭔데? 어떻게 쓰는데? 왜 좋은데?" 전부 여기서 해결합니다.

---

## 1. 이게 뭐야?

쉽게 말하면 **"주식 자동 필터"** 입니다.

매일 주식시장에는 수천 개의 종목이 있어요. 그 중에서 **"오늘 좋은 신호가 나온 종목만"** 골라주는 기계라고 생각하면 됩니다.

### 비유로 이해하기

마치 **유튜브 알고리즘**이 수백만 개 영상 중에서 "나한테 맞는 영상"만 보여주는 것처럼, 이 스크리너는 수천 개 주식 중에서 **"오늘 올라갈 가능성이 있는 종목"** 만 보여줍니다.

## 2. 어떻게 실행해?

### 2-1. 개발 서버 켜기

터미널(검은 창)을 열고 이렇게 입력하세요:

```bash
cd turbo-break
pnpm dev
```

그러면 이런 메시지가 나옵니다:

```
▲ Next.js 16.1.6
- Local: http://localhost:3000
```

### 2-2. 브라우저에서 접속

크롬이나 엣지를 열고 주소창에 입력:

```
http://localhost:3000/login
```

로그인 후 이 주소로 이동:

```
http://localhost:3000/screener
```

짠! 스크리너 화면이 보입니다.

## 3. 뭘 봐야 해?

### 3-1. 핵심만 보자

테이블에서 이것만 보면 됩니다:

| 컬럼 | 의미 | 좋은 값 |
|------|------|---------|
| 종목명 | 회사 이름 | - |
| 등락률 | 오늘 얼마나 올랐나 | +2% ~ +7% (너무 많이 올랐으면 위험) |
| 거래대금 | 오늘 얼마나 거래됐나 | 클수록 좋음 (500억 이상) |
| 조건 통과 ✓ 수 | 10개 중 몇 개 통과? | 10/10 = 가장 안전 |

### 3-2. 조건 10개가 뭐야?

테이블 오른쪽에 작은 ✓/✗ 표시들이 있어요. 이게 10가지 조건입니다.

**쉽게 설명하면:**

1. **N일 돌파** → 최근 N일(1~20일, 선택 가능) 중 가장 높은 가격을 오늘 뚫었냐?
   - 마치 학생이 N일 최고 성적을 오늘 갱신한 것!
   - 동시에 거래량이 평소의 N배(0.5~5배, 선택 가능) 이상이어야 함

2. **횡보** → 최근 N일 동안 가격이 얌전히 있었냐?
   - 롤러코스터가 아니라 잔잔한 파도였냐?

3. **거래량↑** → 오늘 평소보다 N배 이상 사고팔았냐? (기본 2배, 조절 가능)
   - 갑자기 관심이 폭발했다는 신호!

4. **윗꼬리** → 오늘 장마감 가격이 최고가 근처냐?
   - 팔려는 사람이 별로 없다는 뜻

5. **거래대금** → 오늘 500억 이상 거래됐냐?
   - 너무 작은 종목은 위험함

6. **MA60↑** → 60일 평균 가격보다 지금 더 높냐?
   - 전체적으로 오르는 흐름인지 확인

7. **과열X** → 오늘 8% 미만으로 올랐냐?
   - 너무 많이 오른 종목은 내일 위험할 수 있음

8. **양봉** → 시작보다 끝이 높냐?
   - 파는 사람보다 사는 사람이 더 많았다는 뜻

9. **갭X** → 어제보다 갑자기 3% 초과로 시작 안 했냐?
   - 갭은 나중에 다시 내려오는 경우가 많음

10. **5일급등X** → 최근 5일 동안 이미 15% 이상 안 올랐냐?
    - 이미 많이 오른 건 더 오르기 힘듦

### 3-3. 어떤 종목에 관심을 가져야 해?

✓ 10개 = **모두 통과** → 가장 안전하고 좋은 신호
✓ 9개 = **거의 완벽** → 그래도 괜찮음
✓ 8개 이하 = **현재 필터에서 제외됨** (화면에 안 나옴)

> ⚠️ **중요**: 이건 투자 보조 도구입니다. 100% 맞는 게 아니에요.
> 항상 본인이 추가로 확인하고 투자 결정하세요!

## 4. 왜 물개박수를 쳐야 해?

### 기존에 어떻게 했나요?

```
투자자가 하루에 하는 일:
1. 네이버 증권 접속
2. KOSPI 종목 900개 목록 열기
3. 하나씩 차트 열어보기
4. 거래량 확인
5. 이동평균선 확인
6. ...
→ 약 6~8시간 소요
```

### 이제는?

```
1. 브라우저에서 /screener 접속
2. 결과 확인
→ 약 5초 소요
```

**6시간 → 5초** 로 줄었습니다.

### 더 대단한 점

단순히 빠른 게 아니라 **감정 없이** 판단합니다.

- 사람: "이 종목 왠지 오를 것 같은데...?" (감으로 투자)
- 스크리너: "10가지 조건 전부 통과? YES → 표시. NO → 안 표시"

**기준이 항상 똑같습니다.** 기분에 따라 바뀌지 않아요.

## 5. 시장 필터 & CSV 사용법

### 시장 필터

화면 왼쪽 상단 버튼:

```
[ALL] [KOSPI] [KOSDAQ]
```

- **ALL**: 코스피 + 코스닥 전체 보기
- **KOSPI**: 대형주만 보기 (삼성전자, 현대차 같은 큰 회사)
- **KOSDAQ**: 성장주 위주로 보기 (바이오, IT 등)

### CSV 다운로드

오른쪽 상단 `CSV↓` 버튼을 누르면 엑셀로 열 수 있는 파일이 저장됩니다.

```
screener_2026-03-03.csv
```

엑셀에서 열면:
- 모든 종목 데이터
- 10가지 조건 O/X
- 자유롭게 정렬/필터 가능

### 정렬 방법

컬럼 헤더를 클릭하면 정렬됩니다:
- **등락률** 클릭 → 오늘 많이 오른 순서
- **거래대금** 클릭 → 거래가 많이 된 순서
- 다시 클릭 → 반대 방향

## 6. 특정 날짜 결과 보기

URL 주소창에서 날짜를 바꿀 수 있어요:

```
http://localhost:3000/screener?date=2026-03-01
```

`date=` 뒤에 보고 싶은 날짜를 넣으면 됩니다.

## 7. 다음에 뭘 추가하면 더 좋아?

지금 버전(v1)은 기본 기능만 있어요. 앞으로 추가할 수 있는 것들:

### 빨리 하면 좋은 것들

1. **날짜 클릭 선택** → 지금은 URL을 직접 바꿔야 함. 달력 UI 추가하면 편함
2. **실제 주가 연동** → 지금은 가짜 데이터. 실제 한국투자증권 API 연결
3. **모바일 최적화** → 폰에서도 잘 보이게

### 나중에 하면 좋은 것들

4. **알림 기능** → 조건 통과 종목 나오면 카카오톡/슬랙으로 알림
5. **백테스트** → "이 조건으로 과거에 투자했으면 수익률이 얼마였을까?" 계산
6. **즐겨찾기** → 관심 종목 저장
7. **조건 조절** → 8%가 아니라 5%로 바꾸고 싶을 때 UI에서 변경
